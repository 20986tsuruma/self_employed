# <アプリ名>
self_employed

## サイト概要
会社の概要や事業内容の説明と仕事に関する質問や依頼が出来るようにお問い合わせ機能の実装

### サイトテーマ
知らない人が見ても分かりやすいデザインのサイト

### テーマを選んだ理由
私の実家が自営業を営んでおり、プログラミングの勉強を始めた時に両親から会社のHP作成をお願いされたため、PFで作成しようと思いました

### ターゲットユーザ
同業者

### 主な利用シーン
仕事に関する質問や依頼をしたい時
